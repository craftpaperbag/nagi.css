# nagi.css

セマンティックなHTMLに、静けさを。日本語で読むことに最適化された、クラス不要のCSSフレームワーク。

[サンプルを見る](https://craftpaperbag.github.io/nagi.css/)

---

## 特徴

- **クラスレス設計** — `<article>` `<section>` `<h1>` `<p>` など、素のHTMLタグに直接スタイルを当てます。`class=""` を書く必要はありません。
- **日本語最適化** — 行長は `60ch`（日本語で約38文字）、行間は `1.8`。フォントは読み手の環境でもっとも自然に表示される日本語ゴシックに委ねる設計。
- **ダークモード自動対応** — `prefers-color-scheme` に追従します。
- **CSS変数でカスタマイズ** — すべての色・余白・行長は CSS変数 で上書きできます。
- **軽量** — 単一ファイル、約 6KB（未圧縮）。依存なし。

## 導入

### ダウンロード

`nagi.css` をダウンロードして、プロジェクトに配置してください。

## 使い方

クラスを書かずに、セマンティックなHTMLを書くだけです。

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <link rel="stylesheet" href="nagi.css">
</head>
<body>
  <header>
    <h1>タイトル</h1>
    <p>副題や導入文</p>
  </header>
  <main>
    <article>
      <h2>見出し</h2>
      <p>本文。</p>
    </article>
  </main>
  <footer>
    <p>フッター</p>
  </footer>
</body>
</html>
```

## カスタマイズ

CSS変数を上書きするだけで、トーンを調整できます。

```css
:root {
  --nagi-measure: 70ch;        /* 行長を広めに */
  --nagi-accent: #1a5f3f;      /* アクセントを深緑に */
  --nagi-line-height: 2;       /* 行間をさらに広げる */
}
```

主な変数：

| 変数 | 既定値 | 用途 |
|---|---|---|
| `--nagi-measure` | `60ch` | 日本語本文の行長 |
| `--nagi-measure-wide` | `75ch` | 広めの行長（`.wide`クラス使用時） |
| `--nagi-line-height` | `1.8` | 本文の行間 |
| `--nagi-fg` | `#1a1a1a` | 本文の色 |
| `--nagi-bg` | `#fdfdfc` | 背景色 |
| `--nagi-border` | `#e5e3dc` | 罫線の色 |

## 設計原則

1. クラスを書かせない。セマンティックに書くだけで整う。
2. 日本語で読むことを前提に、行長と行間を設計する。
3. 色は最小限、装飾は罫線と余白だけ。
4. 装飾で補強しないと伝わらない情報は、まだ十分に削られていない。

## 立ち位置

nagi.css は、**日本語で読むための classless CSS** です。

行長は日本語の文字数を基準に、行間は長く読み続けられる広さで、フォントは読み手の環境でもっとも自然に表示されるものを尊重する。そういうチューニングを、クラスを書かなくても受け取れるように設計しています。

目指しているのは、**「ちゃんと読める」状態**です。文章に向き合える静けさ。装飾で情報を支えるのではなく、情報が自立できる余白を用意すること。

読み物としてのWebを書く人、読む人のために。
